# KampanyaLab MVP

KampanyaLab, reklam bütçesini kampanya hedefi, sektör, hedef kitle ve süreye göre medya kanallarına dağıtan basit bir medya planlama simülatörüdür.

## Dosyalar

- `index.html` — Ana sayfa ve uygulama arayüzü
- `style.css` — Tasarım dosyası
- `app.js` — Bütçe dağılımı, grafik, rapor ve hesaplayıcı mantığı

## Çalıştırma

Dosyaları bir klasöre çıkarıp `index.html` dosyasını tarayıcıda açman yeterli.

## Yayına alma

Başlangıç için Vercel veya Netlify kullanılabilir. Domain bağlama aşamasında Turhost DNS yönetimine Vercel/Netlify kayıtları girilir.

## Not

Bu MVP tahmini medya planlama simülasyonudur. Gerçek reklam performansı garanti etmez.
